# Skills for opencode-swarm-web-srv

Skills provide specialized instructions and workflows for specific tasks.

## Available Skills

### web-dashboard
Visualization dashboard for swarm agent monitoring with game-like interface.

### realtime-monitoring
WebSocket-based real-time event streaming from .swarm/events.jsonl.

### animation-engine
Smooth CSS/JS animations for agent avatars and task transitions.

## Skill Structure
Each skill directory contains:
- `skill.md` - Main skill instructions
- `resources/` - Supporting files (scripts, templates)
- `examples/` - Usage examples
